최종 수정일: 2026년 5월 4일

임장수첩은 회원가입 없이 사용할 수 있는 현장 임장 기록 앱입니다. 사용자가 입력한 기록은 기본적으로 사용자의 기기 내부에 저장되며, 개발자가 운영하는 서버로 업로드하도록 설계되어 있지 않습니다.

## 운영 주체

- 서비스명: 임장수첩
- 문의: [임장수첩 지원 페이지](https://concept-journey.github.io/imjang/support/)

## 앱이 접근하는 정보

### 위치 정보

- 이용 목적: 현재 위치 표시, 이동 경로 기록, 임장 세션 진행 보조
- 접근 시점: 사용자가 임장 모드에서 위치 기능을 사용할 때

### 카메라

- 이용 목적: 현장 사진 촬영 후 기록 첨부

### 사진 보관함

- 이용 목적: 기존 사진을 기록에 첨부하거나 리포트 이미지를 사진 보관함에 저장

### 사용자가 직접 입력하거나 첨부하는 정보

- 체크리스트 응답
- 텍스트 메모
- 사진
- 음성 파일
- 리포트 이미지

## 저장 방식

사용자가 입력하거나 첨부한 데이터는 기본적으로 사용자의 기기 내부 저장소에 보관됩니다. 앱 삭제 시 기기 내부 데이터는 함께 삭제될 수 있습니다. 사용자가 별도로 저장하거나 외부 앱으로 공유한 파일은 해당 위치에서 별도로 관리해야 합니다.

## 제3자 제공 및 외부 서비스

임장수첩은 개발자 서버로 사용자의 기록 데이터를 전송하도록 설계되어 있지 않습니다. 다만 앱 기능 제공을 위해 다음과 같은 외부 서비스 또는 플랫폼 기능을 사용할 수 있습니다.

- Naver Map SDK: 지도 표시 및 위치 기반 기능 제공
- OpenStreetMap Nominatim: 사용자가 입력한 장소/주소 검색어를 바탕으로 검색 결과 제공
- Apple iOS 시스템 기능: 사진 보관함 접근, 파일 선택, 공유 시트

이들 서비스는 각 제공자의 정책에 따라 별도의 정보 처리를 수행할 수 있습니다.

## 이용자의 선택과 통제

- 사용자는 iOS 설정에서 위치, 카메라, 사진 보관함 권한을 언제든 변경할 수 있습니다.
- 앱 내에서 세션, 포인트, 첨부 파일 기록을 삭제할 수 있습니다.
- 외부로 저장하거나 공유한 결과물은 사용자가 직접 삭제해야 합니다.

## 아동의 개인정보

임장수첩은 아동을 대상으로 설계된 서비스가 아닙니다.

## 방침 변경

본 개인정보처리방침은 서비스 변경 또는 법령 변경에 따라 수정될 수 있습니다. 중요한 변경이 있을 경우 앱 설명 페이지 또는 별도 공지 수단을 통해 안내합니다.
