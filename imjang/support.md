임장수첩은 현장에서 빠르게 임장 기록을 남기고, 나중에 생활권 요약과 리포트로 다시 확인할 수 있게 돕는 iOS 앱입니다.

## 문의하기

앱 사용 중 문제가 있거나 문의가 필요하면 아래 이메일로 연락해 주세요.

- 지원 이메일: [concept.and.journey@gmail.com](mailto:concept.and.journey@gmail.com?subject=%EC%9E%84%EC%9E%A5%EC%88%98%EC%B2%A9%20%EC%A7%80%EC%9B%90%20%EB%AC%B8%EC%9D%98)
- 응답 안내: 영업일 기준 2일 안에 확인 후 답변합니다.

GitHub 계정이 있다면 공개 이슈로도 지원 요청을 남길 수 있습니다.

[GitHub 이슈로 지원 요청하기](https://github.com/concept-journey/concept-journey.github.io/issues/new?title=%EC%9E%84%EC%9E%A5%EC%88%98%EC%B2%A9%20%EC%A7%80%EC%9B%90%20%EC%9A%94%EC%B2%AD)

## 문의 시 포함하면 좋은 정보

가능하면 아래 내용을 함께 적어주세요.

- iPhone 모델
- iOS 버전
- 앱 버전
- 어떤 화면에서 문제가 생겼는지
- 기대한 동작과 실제 동작

Apple ID 암호, 결제 정보, 복구 코드, 상세 주소처럼 민감한 정보는 지원 요청에 포함하지 마세요.

## 지원 범위

- 앱 설치와 실행 문제
- 임장 구역, 지도, 포인트 기록 관련 문제
- 사진, 메모, 파일 첨부 관련 문제
- 리포트 이미지 저장과 공유 관련 문제
- 앱 사용 방법 문의

## 주요 기능

- 임장 구역 생성
- 분위기 임장과 단지 임장 기록
- 지도 기반 동선 확인
- 포인트별 체크리스트, 사진, 메모, 파일 첨부
- 생활권 요약
- 리포트 이미지 저장 및 공유
