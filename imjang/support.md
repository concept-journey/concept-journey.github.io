임장수첩은 현장에서 빠르게 임장 기록을 남기고, 나중에 생활권 요약과 리포트로 다시 확인할 수 있게 돕는 iOS 앱입니다.

## 문의하기

지원 요청은 GitHub 이슈로 남겨주세요.

[지원 요청 작성하기](https://github.com/concept-journey/concept-journey.github.io/issues/new?title=%EC%9E%84%EC%9E%A5%EC%88%98%EC%B2%A9%20%EC%A7%80%EC%9B%90%20%EC%9A%94%EC%B2%AD)

가능하면 아래 내용을 함께 적어주세요.

- iPhone 모델
- iOS 버전
- 앱 버전
- 어떤 화면에서 문제가 생겼는지
- 기대한 동작과 실제 동작

Apple ID 암호, 결제 정보, 복구 코드, 상세 주소처럼 민감한 정보는 지원 요청에 포함하지 마세요.

## 주요 기능

- 임장 구역 생성
- 분위기 임장과 단지 임장 기록
- 지도 기반 동선 확인
- 포인트별 체크리스트, 사진, 메모, 파일 첨부
- 생활권 요약
- 리포트 이미지 저장 및 공유
